Obtener todos los productos (GET):

URL: http://localhost:8080/products
Método: GET
Obtener un producto por su ID (GET):

URL: http://localhost:8080/products/1
Método: GET
Agregar un nuevo producto (POST):

URL: http://localhost:8080/products
Método: POST

{
  "title": "Nuevo Producto",
  "description": "Descripción del nuevo producto",
  "code": "NEW001",
  "price": 15.99,
  "stock": 10,
  "category": "Categoría Nueva",
  "thumbnails": ["new_thumbnail.jpg"]
}

Actualizar un producto por su ID (PUT):

URL: http://localhost:8080/products/2
Método: PUT


{
  "title": "Producto 2 Actualizado",
  "price": 25.99
}


Crear un nuevo carrito (POST):

URL: http://localhost:8080/carts
Método: POST

Agregar un producto a un carrito (POST):

URL: http://localhost:8080/carts/1/product/1
Método: POST
Obtener los productos de un carrito por su ID (GET):

URL: http://localhost:8080/carts/1
Método: GET
